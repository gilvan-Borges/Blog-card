# Blog Preview Card

Projeto de estudo desenvolvido para praticar os fundamentos de **HTML** e **CSS**, recriando um card de blog com design moderno e responsivo.

## Acesse o projeto ao vivo

[https://gilvan-borges.github.io/Blog-card/](https://gilvan-borges.github.io/Blog-card/)

---

## Sobre o projeto

Este projeto consiste em um **card de preview de blog** — um componente visual muito comum em interfaces web. O objetivo foi construir o layout do zero, aplicando conceitos essenciais de estruturação e estilização de páginas, sem o uso de frameworks ou bibliotecas externas.

O card apresenta:
- Uma imagem de destaque no topo
- Uma tag de categoria ("Learning")
- A data de publicação
- Título e descrição do artigo
- Rodapé com foto e nome do autor

## O que foi aprendido

### HTML
- Estruturação de componentes com `div` e hierarquia de tags
- Uso correto de atributos `alt` em imagens para acessibilidade
- Configuração de `meta viewport` para suporte a dispositivos móveis

### CSS
- **Box Model**: controle preciso de `padding`, `margin` e `border` para composição do layout
- **Flexbox**: alinhamento de elementos no rodapé do card com `display: flex` e `align-items`
- **Border Radius**: criação de bordas arredondadas no card e na foto de perfil circular
- **Box Shadow**: efeito de sombra para dar profundidade ao card
- **Media Queries**: adaptação do layout para diferentes tamanhos de tela (responsividade)
- **Pseudo-classes CSS**: efeito de hover no título alterando a cor ao passar o mouse
- **Centralização**: técnica de centralização horizontal e vertical de elementos com `margin: auto`

---

## Layout

O card foi centralizado na página com fundo amarelo (`#f4d04e`), ficando em destaque sobre um fundo branco com sombra preta — técnica usada para simular profundidade e destacar o componente visualmente.
